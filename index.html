02

목표

03

  • 테스트 실패 시 사내 일감 관리 도구에 버그 일감을 자동 생성 — 제목, 재현 경로, 스크린샷까지 포함

  • React / Node.js / TypeScript 풀스택으로 구현하여 브라우저 UI에서 테스트 실행 및 결과 확인 가능
04

결과

  • 일일 접속/업로드 장애 확인 등 반복 운영 체크 공수를 80% 이상 절감
  • 테스트 실패 → 버그 일감 등록까지의 사이클을 완전 자동화
  • Dev / Stage / Live 3개 환경 동시 검증 체계 구축

01

문제 정의

02

목표

  • 이미 번역된 용어의 자동 호출 및 재사용 구조를 구축하여 불필요한 중복 번역 요청 최소화
04

나의 역할
